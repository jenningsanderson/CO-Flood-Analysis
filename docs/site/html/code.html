Retweeted Hashtags¶

rewteet_hashtags.py

This module builds hashtag networks based on retweet frequency. Hashtags are nodes and edges exist between them if they appear in the same tweet. This is similar to a word cloud, but shows relation between tweets.

The hope is that by identifying clusters, SPAM keywords can be easily identified.

retweet_hashtags.make_triangle_cc_plot(graph, threshold=100, show_labels=False)[source]¶

Create a Plot of #Triangles vs. Clustering Coefficient for each node.

Function Options:
Threshold: Maximum # triangles per node to be included in plot.
show_labels (defaults to false): Tells Matplotlib to show labels or not
